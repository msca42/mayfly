# 反馈与触发器
- 振荡器
    - 输出的结果也是输入
    - 能够自动变换
    - 通常采用0和1的交替序列表示输出
    - 输出在0和1之间按照固有规律交替变换
    - 也成为时钟
    - 一个循环(周期): 从某个初始状态开始，经过一段时间又回到先前初始状态的一段间隔
    - 频率: 周期的倒数,一秒内循环的次数,单位是Hz
- 反馈
    - 左边的输入是右边的输出
    - 左边的输出是右边的输入

- 触发器
    - 由两个或非门组成的电路
    - 可以保持信息
    - 存在两个输入，一个是数据端，一个是保持位
    - 保持位的作用是使当前的状态被记住，通常设置为0，此时电路记住最后一次数据端的信息，无论之后数据端如何变化都不会对电路产生影响
