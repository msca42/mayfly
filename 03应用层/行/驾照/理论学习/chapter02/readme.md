## 交通信号
- 交通信号分为交通信号灯、交通标志、交通标线、交通警察手势指挥
- 交通信号灯
    - 红灯(禁止通行)、绿灯(准许通行)、黄灯(警示)
    - 分类:机动车信号灯、非机动车信号灯、人行横道信号灯、车道信号灯、方向指示信号灯、闪光警告信号灯、道路与铁路平面交叉信号灯、掉头信号灯、左转非机动车信号灯
    - 机动车信号灯 
        - 绿灯: 准许车辆通行
        - 黄灯: 警告驾驶人通行时间即将结束、将车停在停止线内、车辆越过停止线的继续通行
        - 红灯: 禁止通行、右转弯车辆可以通行
    - 车道信号灯
        - 绿色箭头和红色叉形组成,设在可变车道上,只对本车道起作用,通常位于收费站入口
    - 方向指示信号灯: 左转、直行、右转、禁止、警示
    - 闪光警告信号灯: 一个黄灯闪烁 注意观察环境,减速慢行
    - 道口信号灯: 两个红灯交替闪烁  红灯熄灭可以通行
    - 非机动车信号灯: 红黄绿三色,与机动车信号灯相同
    - 人行横道信号灯: 红绿两色, 红灯时已经在人行横道的在确认安全时可以继续通过
    - 非机动车信号灯: 左转、右转、禁止、直行
    - 掉头信号灯: 不能妨碍其他车辆通行
- 交通标志
    - 以颜色、形状、字符、图形为标志的道路管理信息
    - 分为主标志和辅助标志
    - 主标志: 警告标志、禁令标志、指示标志、指路标志、旅游区标志、作业区标志、告示标志
    - 警告标志
        - 黄底黑边黑图案、等边三角形或矩形
        - 交叉路口警告标志
            - 十字: 前方是十字路口
            - T型: 前方是T型路口
            - 环型: 前方是环形路口
            - Y型: 前方路口交叉角小于90度
        - 弯道警告标志
            - 急弯路标志: 向左向右急弯
            - 反向弯道: 车速限制40km/h
            - 连续弯道
        - 陡坡警告标志
            - 上坡标志、下坡标志、连续下坡标志
        - 窄路窄桥和双向交通警告标志
            - 两侧变窄、左侧变窄、右侧变窄
            - 窄桥标志
            - 双向交通:前方是不分离双向行驶路段、应注意会车
        - 注意行人和儿童警告标志
        - 注意牲畜和野生动物警告标志
        - 注意信号灯和落石警告标志
        - 注意村庄和傍山险路警告标志
        - 易滑和堤坝路警告标志
        - 注意横风、隧道和驼峰警告标志
        - 渡口和复杂路面警告标志
            - 渡口标志、过水路面标志、路面不平标志、路面高突标志、路面低洼标志
        - 铁路道口的相关警告标志
            - 有人看守、无人看守、叉形、距离铁路道口
        - 注意非机动车和残疾人警告标志
        - 事故易发路段、慢行和注意危险警告标志
        - 注意障碍物和前方车辆排队警告标志
            - 左侧、右侧、左右绕行
        - 施工、建议速度、隧道开灯和注意潮汐车道警告标志
        - 注意保持车距、分离式道路和合流警告标志
        - 注意不同气候和路面的警告标志
            - 结冰、雾天、雨雪、不利气象
    - 禁令标志
        - 加以限制、加以禁止、白底红圈红杠黑图案
        - 通行和禁止相关的禁令标志
            - 停: 停车瞭望、确认安全后通行
            - 让: 减速慢行或停车观察
            - 会车让行: 会车时必须停车,让对方先行
            - 禁止通行: 空白红圈 
            - 禁止驶入: 红横杠
            - 禁止车辆临时或长期停放: 蓝底双叉
            - 禁止车辆长期停放: 蓝底单叉
            - 停车检查
            - 海关
            - 禁止运输危险物品
        - 禁止机动车驶入的禁令标志
            - 禁止机动车驶入
            - 禁止载货汽车驶入
            - 禁止电动三轮车驶入
            - 禁止大型客车驶入
            - 禁止小型客车驶入
            - 禁止挂车半挂车驶入
            - 禁止各类拖拉机驶入
            - 禁止摩托车驶入
            - 禁止三轮汽车、低速货车驶入
            - 禁止标志所示车辆驶入
        - 禁止非机动车驶入
            - 禁止各类非激动驶入
            - 禁止畜力车驶入
            - 禁止人力客运三轮车驶入
            - 禁止人力货运三轮车驶入
            - 禁止人力车进入
        - 行人和车辆中的禁令标志
            - 禁止行人进入
            - 禁止一切车辆向左
            - 禁止一切车辆向右
            - 禁止一切车辆直行
            - 禁止一切车辆向左右
            - 禁止一切车辆直行和向左
            - 禁止一切车辆直行和向右
            - 禁止一切车辆掉头
            - 禁止车辆鸣喇叭
            - 限制宽度
            - 限制高度
            - 限制质量
            - 限制轴重
        - 禁止和解除禁令标志
            - 禁止超车、解除禁止超车
            - 限速、解除限速、区域限速、区域限速解除
            - 区域禁止长期停车、区域禁止长期停车解除
            - 区域禁止停车、区域禁止停车解除
    - 指示标志
        - 蓝底白图、圆形、矩形、正方形
        - 与行驶方向有关的指示标志
            - 直行、向左、向右
            - 直行向左、直行向右、向左向右
            - 右侧、左侧行驶
            - 立体交叉直行和左转弯、立体交叉直行和右转弯
            - 环岛行驶
            - 单向向左、向右
        - 指导驾驶人驾驶行为的指示标志
            - 鸣喇叭
            - 最低限速
            - 会车让行、会车先行
            - 路口优先通行
        - 车行道行驶方向的指示标志
            - 右转、左转、直行车道
            - 掉头车道、向左和掉头
            - 直行和左转合用、直行和右转合用
            - 分向行驶
        - 专用道路、车道和停车场指示标志
            - 机动车行驶(车道)
            - 非机动车行驶(车道)
            - 公交线路专用道(快速公交系统)
            - 多乘员车辆专用车道
            - 人行横道、步行道
            - 允许掉头
            - 停车交通标志: P 允许机动车停放
    - 指路标志
        - 为驾驶人提供道路方向、地点、距离等信息，
        - 一般道路:蓝底白图白边框蓝色衬边
        - 高速公路和城市快速路: 绿底白图形白边框绿色衬边
        - 道路编号标志
            - 公路等级: G国道(高速+普通)、S省道(高速+普通)、X县道、Y乡道、村道、专用公路
            - 技术等级: 高速公路、一级公路、二级公路、三级公路、四级公路
        - 道路设施标志
            - 错车道、应急避免场所
        - 道路及设施预告标志
            - 此路不通、交通监控设备、车道数减少增加
        - 线形诱导标志
            - 1～3个单元: 引导行车方向、提示前方线形变化
            - 两侧、右侧、左侧通行
    - 其他标志
        - 旅游区标志
        - 作业区标志
        - 道路施工标志
        - 告示标志
- 交通标线
    - 按形态： 线条、字符、凸起路标、轮廓标
    - 指示标线
        - 纵向
            - 黄色虚线 可跨越对向车道分界线
            - 白色虚线 可跨越同向行驶的分界线
            - 双黄虚线 潮汐车道
            - 车行道边缘白色实线 划分机动车道和非机动车道、不可跨越
            - 车行道边缘白色虚线 划分机动车道和非机动车道、可临时跨越
            - 车行道边缘白色虚实线 划分机动车道和非机动车道, 虚线侧可以跨越
            - 黄色单实线边缘线 机动车单向行驶、非机动车双向行驶 不能跨越
        - 横向
            - 人行横道
            - 人行横道预告标识
            - 白色折线 车距确认
        - 其他标线
            - 平行式停车位
            - 倾斜式停车位 
            - 垂直式停车位
            - 固定停车方向停车位
            - 蓝色表示免费、白色表示收费、黄色专属
    - 禁止标线
        - 纵向
            - 双黄实线: 禁止跨越对向车道
            - 双黄虚实线: 实线禁止跨越，虚线可以临时跨越
            - 黄色单实线: 禁止跨越
            - 白色实线: 禁止跨越
            - 黄色虚线边缘: 禁止长时间停车 
            - 黄色实线边缘: 禁止停车 
        - 横向
            - 白色单实线: 在其后停车
            - 白色双实线: 礼让干道
            - 禁止掉头
    - 警告标线
        - 纵向
        - 横向
        - 其他
- 交通警察手势
