# 绪论

## 算法的作用

- 算法:完成一项任务所遵循的一系列步骤
- 程序: 算法在机器上的兼容形式
- 程序设计/编程: 开发程序，采取与机器兼容的形式进行编码并将其输入机器的过程
- 软件：程序及其所表示的算法
- 硬件: 机械

- 一旦找到执行任务的算法，执行该任务时就不再需要了解该算法所依据的原理
- 找不到解决问题的算法，机器就解决不了问题
  - 机器的智能水平受限于算法所传达的智能
