# 数据压缩

## 目标
- 区分无损压缩和有损压缩
- 描述游程长度编码和它是如何实现压缩的
- 描述赫夫曼编码和它是如何实现压缩的
- 描述 Lempel Ziv 编码以及字典在编码和译码中的作用
- 描述压缩静止图像的 JPEG 标准背后的主要思想
- 描述压缩视频的 MPEG 标准背后的主要思想以及它与 JPEG 间的关系;
- 描述压缩音频的 MP3 标准背后的主要思想


## 引言
- 数据压缩意味发送或存储更少的位数，可以分为两类: 无损压缩和有损压缩
- 无损压缩
    - 游程长度编码
    - 赫夫曼编码
    - LZ编码
- 有损压缩
    - 图像压缩JPEG
    - 视频压缩MPEG
    - 音频压缩MP3
