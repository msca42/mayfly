# 编程语言发展史
- 计算机只能处理二进制数字
- 机器语言: 二进制数字的集合
- 汇编语言: 使用助记符表示二进制指令    
    - 通过汇编器将汇编语言转化为机器语言
    - 汇编语言和机器语言一一对应
- 高级语言
    - 相对于机器语言和汇编语言而言，前者是低级语言
    - 通过编译器可以将高级语言转化为低级语言
    - 不需要考虑寄存器和内存地址，内存地址抽象为变量
    - Fortran
        - 最初只能运行在IBM电脑上
    - COBL
        - 能够运行在所有机器上
        - 实现机制: 
            - 每个计算架构都需要一个COBOL编译器 
            - 可以接收相同的代码
            - 一次编写到处运行
    - ALGOL、LISP、BASIC
    - Pascal、C、Smalltalk
    - C++、Object-C、Perl
    - Python、Ruby、 Java
    - Swift、C#、 Go
