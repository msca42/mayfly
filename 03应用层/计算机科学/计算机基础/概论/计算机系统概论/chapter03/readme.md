# 数字逻辑

## MOS 晶体管

- MOS 管可以分为 p 型和 n 型，机制互补
- n 型 MOS 管
  - 结构: 漏极 D 栅极 G 源极 S
  - 特性：
    - 栅极接入 2.9V 电压，则在源极和漏极之间产生通路
    - 栅极电压是 0V，则源极和漏极之间断开
- p 型 MOS 管
  - 结构：源极 S 栅极 G 漏极 D
  - 特性
    - 栅极接入 2.9V 电压，则在源极和漏极之间断开
    - 栅极电压是 0V，则源极和漏极之间导通
- 如果电路同时包含 P-MOS 和 N-MOS 晶体管，称该电路为 CMOS 电路

## 逻辑门

- 使用 CMOS 电路实现与或非异或逻辑运算，该电路称为逻辑门电路
- 非门
