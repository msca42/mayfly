# JavaScript 简介

- JavaScript 是一门高级、动态、解释型编程语言，非常适合面向对象和函数式编程风格
- 每种语言都必须有一个平台或标准库，用于执行包括基本输入和输出在内的基本操作
- 核心 JavaScript 语言
  - 定义了最小限度的 API，包含可以操作数值、文本、数组、集合、映射的等
- 运行环境
  - 通常包含输入输出等基本操作
  - JavaScript 的宿主环境有两个，一个是浏览器，一个是 Node
    - 浏览器允许 JS 从用户的鼠标和键盘或通过发送 HTTP 请求获取输入，也允许 JS 通过 HTML 和 CSS 向用户显示输出
    - Node 允许 JS 读写文件、通过网络发送和接收数据、以及发送和处理 HTTP 请求

## 探索 Javascript

- 在浏览器和 node 中如何运行 JavaScript 代码
  - 浏览器
    - Windows: ctrl + shift + i -> 控制台
  - Node
    - 下载 Node.js，在命令行窗口输入 node
