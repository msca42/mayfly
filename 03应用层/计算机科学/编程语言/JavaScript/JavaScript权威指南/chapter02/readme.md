# 词法结构

- 词法结构是一套规则，规定了一门语言的最低级的语法

## Javascript 程序的文本

- 区分大小写，意味着关键字、变量、函数名和其他标识符始终保持一致的大小写形式
- 忽略程序记号之间的空格，也忽略换行符
  - 这里的空格包含常规空格(\u0020)， 制表符、各种 ASCII 控制符和 Unicode 间隔

## 注释

- 单行注释 //
- 多行注释 /\*\*/

## 字面量

- 字面值是一种直接出现在程序中的数据值

## 标识符和保留字

- 标识符
  - 是一个名字
  - 用于命名常量、变量、属性、函数和类，以及为某些循环提供标记(label)
  - 由字母、下划线、美元符号和数字组成、数字不能是第一个字符
- 保留字
  - JavaScript 为语言自身使用而保留的标识符、不能作为常规标识符使用
  - 这里的保留字既包含关键字，也包含保留字

## Unicode

- 标识符中通常使用 ASCII 字母和数字，但是可以支持 Unicode 字母
- Unicode 转义序列
  - 转义序列可以仅使用 ASCII 字符来表示 Unicode 字符
  - 使用\u 开头，后跟 4 位十六进制数字
  - 可以出现在字符串字面量、正则表达式字面量和标识符中
  - ES6 中可以使用花括号的版本，用于支持大于 16 位的 Unicode 码点
- 归一化
  - Unicode 标准为所有字符定义了首选编码并规定了归一化例程

## 可选的分号

- JavaScript 使用分号分隔语句，但是也可以省略作为一种写法
