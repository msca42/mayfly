# 创建 HTML 文档

## 构筑基本的文档结构

- 文档元素
  - 确定文档的轮廓以及浏览器初始环境
  - 只有四个
    - DOCTYPE 元素
      - 处理的是 HTML 文档
      - 用来标记文档内容的 HTML 版本
    - html 元素
      - 根元素
      - 文档的开始
    - head 元素
      - 包含元数据和文档信息
      - 元数据向浏览器提供有关文档内容和标记的信息
    - body 元素
      - 包含文档内容

## 用元数据元素说明文档

- 设置文档标题
  - title 元素
    - 设置文档的标题或名称，浏览器将内容显示在窗口顶部或标签页上
- 设置相对 URL 的解析基准
  - base 元素
    - 设置基准 URL，让 HTML 从相对链接开始解析
- 用元数据说明文档
  - meta 元素
    - 定义文档的各种元数据
    - name 和 content 属性用名/值定义元数据
    - 声明字符编码 charset
    - 模拟 HTTP 标头字段
- 定义 CSS 样式
  - style 元素
    - 定义 HTML 文档内嵌的 CSS 样式
    - media 属性：指定文档在什么情况下使用定义样式
      - 属性值为设备类型
      - 可以通过逻辑运算符来整合不同设备和限定特性
- 指定外部资源
  - link 元素
    - 常见属性: href、rel
