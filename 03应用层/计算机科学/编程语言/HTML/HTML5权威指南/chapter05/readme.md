# 初探 JavaScript

## 准备使用 JavaScript

- 在 HTML 中使用脚本的方法
  - 内嵌脚本
  - 外部脚本

## 使用语句

- JS 的基本元素是语句，通常以分号结尾，但可以省略

## 定义和使用函数

- 将语句包装在函数中，可以在调用函数时，才会执行语句
- 定义带参数的函数
  - JavaScript 时弱类型语句，定义函数时不必声明参数类型
  - 调用函数时提供的参数不必和定义中的参数相同，如果更少，未提供的值为 undefined，如果更多，忽略多出的值
- 定义会返回结果的函数

## 使用变量和类型

- 定义变量使用 var 关键字
  - 局部变量：定义在函数中的变量
  - 全局变量：直接在 script 元素中
- 弱类型语言
  - 不同明确声明变量的类型就可以使用变量表示不同类型的值
- 使用基本类型
  - 基本类型包括字符串类型、数值类型和布尔类型
  - 字符串类型
    - 用引号包裹一串字符表示
  - 布尔类型
    - 只有 true 和 false 两个值
  - 数值类型
    - 使用 number 类型标识
- JavaScript 支持对象概念
  - 对象具备属性和方法，属性指属性名和属性值，方法指
  - 创建对象
    - 方式一： new Object()
    - 方式二: 键值对字面量
      - 键表示属性，值表示属性对应的值，使用冒号间隔
      - 值可以表示具体的数值，也可以是函数
        - 对象的函数称为方法，添加值为方法的属性也称为添加函数
  - 使用对象
    - 读取和修改对象属性值
      - 通过对象名.属性名或对象名["属性名"]实现
    - 枚举对象属性
      - 使用 for-in 语句
        - 每次迭代过程中，属性名会被赋予中间变量
    - 增删属性和方法
      - 通过新增键值对的形式可以增加属性，属性为函数时，新增为方法
      - 通过关键字 delete 可以删除对象的属性
    - 判断对象是否具备某个属性
      - 通过关键字 in 实现

## 使用 JavaScript 运算符

- 相等和等同运算符
  - 相等
    - 尝试将操作数转换为同一类型
  - 等同
    - 值和类型都相同
  - JS 基本类型的比较是值的比较，对象的比较是引用的比较，也就是地址值的比较
- 显式类型转换
  - 通过+可以将不同类型和字符串进行连接，此时不同类型都转换为字符串
  - 将数值转换为字符串的方式
    - ().toString() 方法
    - String() 函数
  - 将字符串转换为数值
    - Number()函数

## 使用数组

- JS 会自动调整数组中可容纳的元素，且不区分类型
- 创建数组方式
  - 对象
  - 字面量
- 读取和修改数组内容
  - 通过方括号和索引实现读取和修改
- 枚举数组内容
  - 通过 for 循环枚举数组中的内容

## 处理错误

- 使用 try-catch 处理错误

## 比较 undefined 和 null 值

- undefined
  - 读取未赋值的变量或对象没有的属性时的值
- null
  - 赋值但是该值不是一个有效值
- 检查变量或属性是否为 undefined 或 null
- 区分 null 和 undefined
  - 如果想同等对待，使用==，JS 会将 undefined 转换为 null
  - 如果要区分对象，使用===
