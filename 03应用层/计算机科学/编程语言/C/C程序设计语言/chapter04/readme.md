# 函数与程序结构
## 函数的基本知识
- 格式: 
    - 返回值类型 函数名(参数声明表) {
        声明和语句
    }
- 默认返回值类型为int，可以省略
- 程序可以看作变量定义和函数定义的集合
    - 函数之间通信通过参数、函数返回值以及外部变量进行
    - 返回值通过return传递给调用函数，返回值可以是表达式，经过类型转换变成返回值类型
## 返回非整数值的函数
-  需要提前声明函数，并且声明类型与返回值类型一致

## 外部变量
- 外部变量和函数具有的性质: 
    - 通过同一个名字引用的所有外部变量实际上都是引用同一个对象
- 作用域与生命周期 
    - 可以在全局范围内访问，所以通常作为函数数据交换的媒介
    - 随着程序的加载而产生，随着程序的结束而终结
