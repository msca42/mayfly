# 初识 C 语言

- 主要内容
  - C 的历史和特性
  - 编写程序的步骤
  - 编译器和链接器的一些知识
  - C 标准

## 起源

- 1972 年，丹尼斯·里奇和肯·汤姆森在开发 Unix 系统时设计了 C 语言

## 优点

- 高效性
  - 具备汇编语言才具有的微调控能力
    - 汇编语言是为特殊 CPU 设计的指令，使用助记符表示
    - 不同架构的 CPU 使用不同的汇编语言
- 可移植性
  - C 程序只要稍作修改或不修改就可以在其他系统运行
  - 使用 C 编译器可以将 C 源代码转换为汇编指令

## 应用范围

- 操作系统、计算机游戏、嵌入式系统、机器人、计算机语言
