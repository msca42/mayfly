# 字典和结构化数据
## 字典数据类型
- 字典是键值对的集合，使用{}表示，通过键访问值
- 字典是无序的
- 方法
    - keys(): 返回所有键 类型dict_keys
    - values(): 返回所有值 类型dict_values
    - items(): 返回键值对 类型dict_items
    - get(): 检查键是否在字典中, 参数1要取得的键，参数2该键不存在时返回的备用值
    - setdefault(): 第一个参数时要检查的键，第二个参数是当该键不存在时要设置的值，如果该键存在，返回键的值
## 美观地输出
- pprint模块
    - pprint()
    - pformat() 以字符串输出
