## 创建数值列表
- Python用于处理数值列表的工具很多
- range()函数
    - 参数: start、end、step
    - 生成一系列的数,从start到end-1
    - 类型range
    - 通过内置函数list()可以转化为列表
- 统计函数
    - min()
    - max()
    - sum()
- 列表推导式
    - 一行代码生成列表
    - 将for循环和创建新元素的代码合并
    - 组成
        - 表达式: 用于生成要存储到列表中值
        - for循环
        - 判断条件

## 使用列表的一部分
- 切片: 处理列表的部分元素

    - [:]可以复制列表
