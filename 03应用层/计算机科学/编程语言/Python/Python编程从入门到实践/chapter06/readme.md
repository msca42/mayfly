# 字典

## 一个简单的字典

## 使用字典
- 通过键作为索引可以实现以下功能
    - 访问值
    - 添加键值对
    - 修改值
- 通过del可以删除键值对

- 当访问不存在键的值默认会报
    - 解决方案: get()
    - 在指定的键不存在，返回一个默认值
    - 参数: 指定键、键不存在时返回的值

## 遍历字典
- items(): 返回所有键和值
- keys(): 返回所有键，默认
- values(): 返回所有值
-----
- 集合Set
    - 所有元素具有唯一性，剔除重复项
    - 内置函数set()
    - 可以使用{}包裹元素
## 嵌套
- 将字典存储在列表或将列表存储在字典中
