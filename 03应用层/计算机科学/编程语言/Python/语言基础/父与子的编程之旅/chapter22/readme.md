## 文件
- 计算机按照二进制格式来存储信息，只用1和0来表示信息
- 每8位组成一组，称为1字节
- 文件具有4个属性
    - 名字、类型、位置、大小

## 文本文件和二进制文件
- 文本文件: 包含文本信息，包括字母、数字、标点符号和一些特殊字符
- 二进制文件: 不包含文本信息，但可能包含音乐、图片或其他类型的数据
- 参数
    - 读文件: r
    - 写文件: w
    - 追加文件: a
