# 异常
## 异常与分类
- 异常: 在解释器执行过程中出现错误的一种管理机制
- 分类
    - 基类: 
        - BaseException 内置异常
        - Exception 内置非系统退出类异常

## 捕获异常
- 格式
    - try:
        - 可能产生异常的代码
    - except 异常类型: 
        - 处理异常
    - else:
        - 没有出现异常，执行此部分代码
    - finally:
        - 无论异常，都会执行

## 自定义异常
- 创建自定义异常类  
- 在try中抛出异常raise 异常类
