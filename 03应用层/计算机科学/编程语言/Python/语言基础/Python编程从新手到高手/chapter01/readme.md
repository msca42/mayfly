# Python 的哲学

- 理解 Python 是什么，从 Python 的哲学开始是学习 Python 的最好路径

## 什么是 Python

- 一门高级通用编程语言，支持过程式编程、面向对象编程和函数式编程
- 如果要真正掌握，必须从它本身开始，而非和其他语言进行比较

## Python 不是什么

- 脚本语言
  - Python 是一门图灵完备的语言
    - 可以用 Python 实现任何其他编程语言
    - 任何其他编程语言可以做的事情，Python 都可以做
- 解释型语言比编译型语言慢
  - 默认解释器 CPython 在执行效率上比原生机器码低，但是其他库、解释器实现整体性能都有显著提高，接近原生机器码的效率
  - Python 的性能足够作为应用开发、数据分析、科学计算等领域的首选语言
- Python 不能被编译
  - 在某些情况下，Python 代码可以转换为 C，Java 等其他语言，继而实现编译
- Python 在后台编译
  - 将原始代码编译位稍后执行所需的字节码，解释器包含一个虚拟机，如同 CPU 执行机器码一样执行 Python 字节码
- Python 不适合大型项目
  - Python 擅长将大项目变成多个简单的小项目

## 定义"Pythonic"代码

- 能较好利用语言本身功能的惯用代码
- 一种哲学: 只有一种方法可以做到这一点
  - 对于任何特定问题，都有单一的，可量化的最佳解决方案
