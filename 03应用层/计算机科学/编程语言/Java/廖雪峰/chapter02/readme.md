# 快速入门

## Java 历史

- Sun 公司的詹姆斯·高斯林开发
- 语言类型
  - 编译型
    - 典型语言：C、C++
    - 将源代码编译成机器码
    - 由于平台(x86,arm)的 CPU 指令集不同，机器码也不同
  - 解释型
    - 典型语言：Python、Ruby
    - 逐行解释运行源代码
  - 半编译半解释
    - 将源代码编译生成字节码，将字节码加载进 JVM，半解释半编译执行
- 版本
  - JavaSE 标准版
    - 包含 JVM 和标准库
  - JavaEE 企业版
    - 包含 JavaSE、API 和库
  - JavaME 微型版
    - 针对嵌入式设备
- 名词解释
  - JDK: Java 开发工具包
    - 包含 JRE 和编译器、调试器等开发工具
  - JRE：Java 运行时环境
    - 包含 JVM 和运行时标准库
  - JSR：Java 提案文档
    - 与 JSR 一同发布的是参考实现 RI 和兼容性测试套件 TCK
  - JCP: 管理和制定 Java 技术规范的组织

## 搭建开发环境

- 安装 JDK
- 配置环境变量
  - window 设置 JAVA_HOME,PATH
- Helloworld
  - class 表示类，类名为 Hello、public 表示公开的
  - 方法：可执行的代码块
  - main 方法:
    - public 表示公开的，static 表示静态的，void 表示返回值类型，{}表示方法的代码
  - System.out.println()表示 out 对象的 println()方法，意为换行打印字符串
  - 是程序的入口函数
- 运行 Java 程序
  - 编写源代码
  - 编译生成字节码
    - javac Hello.java
  - 在 JVM 中加载运行
    - java Hello
- 注意事项
  - 一个 Java 类只能定义个 public 类型的 class，且 class 名与文件名一致

## Java 程序基础

- Java 程序基本结构
  - 基本单位是 class，类名是 Hello
  - 类名的命名要求
    - 英文字母大写开头，后接字母、数字和下划线的组合
  - public 表示公开的、class 内部有若干方法
  - 方法是一组执行语句，按照顺序执行的方式执行
    - main 是方法名，void 表示没有任何返回值
    - 方法名的命名要求
      - 同类名，区别是首字母小写
  - 语句是最基本单位，以分号结束
  - 注释是给程序员理解代码的，编译器会自动忽略
    - 可以分为单行注释，多行注释，文档注释
- 变量和数据类型

  - 变量
    - 使用
      - 先定义后使用
      - 定义的时候可以赋值一个初始值
      - 可以重新赋值
    - 分类
      - 基本类型：CPU 可以直接运算的类型
        - 整数: byte short int long
          - 占用内存字节数: 1 4 8 16
          - 表示范围: -2^(n-1) ~ 2^(n-1) - 1
        - 浮点数 float double
          - 就是小数，因为使用科学计数法表示时，小数点可以浮动
        - 字符 char
          - 表示一个字符，可以表示一个标准的 ASCII 或一个 Unicode 字符
        - 布尔 boolean
          - 只有 true 和 false 两个值
          - 理论上只要 1 比特，但通常表示为 4 字节
      - 引用类型
        - 类似 C 语言的指针，内存存储了一个地址，指向对象在内存中的位置
  - 常量
    - 使用 final 修改的变量
    - 在一次初始化之后无法再次赋值
    - 通常全部大写
  - var
    - 可以省略变量类型，之后根据赋值语句推断变量类型
  - 作用域
    - 通常变量作用域就是包裹变量的代码块，从变量定义开始，到语句块结束

- 整数运算

  - 遵循四则运算规则
  - 整数运算都是精确的，整数除法的结果也是整数
  - 运算符: + - \* / %
  - 溢出: 计算超出范围，返回一个奇怪的结果
  - 复合赋值运算符
    - 将整数运算符和赋值运算符组合
    - 运算符: += -= \*= /=
  - 自增、自减
    - 运算符: ++ --
    - 表示对变量进行+1 或-1 操作
  - 移位运算
    - 左移: 符号位改变，相当于乘以 2，超出范围溢出
    - 右移: 符号位不变，相当于除以 2
    - 无符号右移: 符号位置零
  - 位运算
    - 对位进行与、或、非和异或运算
  - 运算符优先级
    - 加括号
  - 类型自动提升与强制转型
    - 参与运算的数据类型不一致，计算结果为较大类型
    - 通过(类型)可以实现强制类型转换

- 浮点数运算
  - 无法精确表示
  - 判断两个浮点数是否相等，一般是判断两个浮点数之差的绝对值是否小于一个很小的数
  - 类型转换
    - 浮点数和整数运算，通常整型提升为浮点类型
  - 溢出
    - 整除除以 0 报错，浮点数除以 0，返回特殊值
    - NaN 表示不是一个数字
    - Infinity 表示无穷大
    - -Infinity 表示负无穷大
- 布尔运算
  - 只有 true 和 false 两个值
  - 包含
    - 比较运算符 > >= < <= == !=
    - 与运算 &&
    - 或运算 ||
    - 非运算 !
  - 短路
    - 如果一个布尔运算的表达式提前确定结果，则后续的计算不再执行，直接返回结果
  - 三元运算符
    - b?x:y
- 字符和字符串

  - 字符类型 char
    - 基本数据类型，可以保存一个 Unicode 字符
    - 占用两个字节
  - 字符串类型 String
    - 由双引号表示字符串，可以存储 0 个到任意个字符
    - 特点
      - 不可变(字符串内容不可变)
    - 常见转义字符
      - \n 换行、\r 回车、\t Tab
      - \u#### 表示一个 Unicode 编码
    - 使用+，可以连接字符串和其他字符，也可以进行字符串之间的连接
    - 使用""""""表示多行字符串
    - null 值
      - 表示不存在，即变量不指向任何对象

- 数组类型
  - 用来表示一组相同类型的值
  - 声明:
    - 类型[] 数组名；
  - 初始化方案
    - 动态初始化：new 类型名[数组容量];
    - 静态初始化: new 类型名[]{元素 1，元素 2...};
      - 可以简化
      - 简化之后的右值，只能用于初始化时，不能用于赋值语句
      - 简化之前的右值可以用于赋值语句
  - 特点
    - 所有元素初始化为默认值，整型 0，浮点型 0.0，布尔型 false
    - 数组一旦创建，大小不可改变
  - 获取数组信息
    - 访问数组元素
      - 通过索引访问，从 0 开始
    - 获取数组长度
      - length 属性

## 流程控制

- 输入与输出
  - 输出
    - System.out 标准输出流
    - println() 换行输出
    - print() 不换行输出
    - printf() 格式化输出
  - 输入
    - Scanner 类 和 System.in 标准输入流
    - nextLine()：读取一行输入并获取字符串
    - nextInt()：读取整数
- if 条件判断
  - 格式: if(condition){ statement } else if(condition){ statement} else{
    statement
    }
  - == 运算符
    - 判断标准类型表示值是否相等
    - 判断引用类型表示是否是同一个对象
- switch 多重选择
  - 格式 1：switch(variable) { case variable : statement1; break; default: statement2; break;}
    - 当 case 没有 break，switch 具有穿透性
  - 格式 2：switch(variable) { case variable -> statement1; default -> statement2}
    - -> 在遇到多条语句中，使用{}括起来，不需要写 break，没有穿透性
    - statement 可以变更为数值返回
  - yield 关键字可以返回 switch 的返回值
- while 循环
  - 格式: while(条件表达式) {循环语句}
- do while 循环
  - 格式: do {执行循环语句} while(条件表达式);
- for 循环
  - 格式: for(初始化;循环条件;步进){statement}
  - for each 循环
- break 和 continue
  - break 跳出一层循环
  - continue 中断当前循环继续下一次循环

## 数组操作

- 遍历数组
  - 通过 for 循环可以遍历数组
  - 通过 Arrays.toString()可以快速打印数组内容
- 数组排序
  - 常见排序: 冒泡排序、插入排序、快速排序
  - 冒泡:
    - 每一轮循环，将最大的数交换到末尾
  - Java自带排序Arrays.sort()
- 多维数组
  - 数组中存在数组
  - 遍历  
    - 通过双层for循环
    - Arrays.deepToString()
- 命令行参数
  - 通过String[] 数组接收
