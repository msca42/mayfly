# 其他表单元素及输入验证

## 使用其他表单元素
- select
    - 生成一个选项列表
- optgroup
    - 对option元素进行编组
- textarea
    - 生成多行文本框
- output
    - 表示计算的结果
    - 通过for属性链接两个input元素的id
- keygen
    - 生成公开/私有密钥对
    - 公钥发给服务器、私钥由浏览器保存并存入用户密钥库

## 使用输入验证
- 输入验证通过表单元素的一些属性控制，常见属性required、min、max、pattern

## 禁用输入验证
- form元素的novalidate属性或button、input元素的formnovalidate属性
