## 前言

* 什么是编程
    * 准确的告诉计算机要执行的操作的过程
    * 要执行的操作也称为**指令**
    * 程序是由指令组成
    * 软件就是运行的程序
* 计算机底层的硬件实现
    * 大量的电路组成，通过记录电路开闭的状态来表示
    * 电路的开闭使用二进制0和1表示
    * 单个二进制称为一位
* 人类如何和计算机沟通
    * 计算机只能识别二进制指令
    * 人类创建可阅读编程语言
    * 然后将编程语言翻译为二进制代码
* 选用Python
    * 免费、开源、具有工业性、跨平台

## 收集起来--列表与字典

* 数据结构
    * 数据在程序中的组织方式
* 列表
    * 使用中括号[]包裹元素，使用逗号分隔元素
    * 内置函数
        * list()
    * 特点
        * 可以包含任何内容作为元素
        * 有序集合
        * 可变
    * 获取元素
        * 获取单个元素：使用角标
        * 获取多个元素：切片
    * 添加元素
        * append() 末尾添加单个元素
        * extend() 末尾添加多个元素
        * insert() 某个位置插入元素
    * 删除元素
        * remove() 删除选中元素
        * del 利用角标删除元素
        * pop() 删除列表最后一个元素或者删除指定位置元素
    * 修改元素
        * 使用角标
    * 查找元素
        * in 元素是否在列表中
        * index() 元素在列表中具体位置
    * 排序
        * sort()
        * reverse() 逆序排列
        * sorted() 函数
* 元组
    * 使用圆括号()包裹元素，使用逗号分隔元素
    * 内置函数
        * tuple()
    * 特点
        * 不可变
* 字典
    * 使用大括号{},以键值对的形式组织
    * 特点
        * 值可以包含任何内容
        * 键只能是不可变量
    * 列出所有键
        * keys()
    * 列出所有值
        * values()
    * 删除
        * 单个元素
            * del
        * 清空字典
            * clear()
    * 判断字典是否存在某个键
        * in

## 函数

* 随着代码的复杂度越来愈高，通常基于以下三个维度拆分程序
    * 函数：可以重复使用
    * 对象：把程序的各个部分编写为自包含的单元
    * 模块：包含程序各个部分的独立文件
* 定义函数
    * def + 函数名
* 调用函数
    * 函数名()
* 向函数传递参数
    * 函数定义中的参数称为形参
    * 函数调用中的参数称为实参
* 返回值
    * 函数返回的数据
* 变量作用域
    * Python具有内存管理机制
    * 局部变量
        * 修改局部变量为全局变量
            * 关键字global
    * 全局变量
        * 可以在函数内访问和使用
        * 如果修改，则会创建新的局部变量

## 对象

* 属性：对象的特征
    * 访问
        * 对象名.属性名
* 方法：对象的函数
* 对象 = 属性 + 方法
* 创建对象
    * 创建类
        * 初始化对象
            * __init__()
        * 创建方法
    * 使用类创建对象
        * 对象名 = 类名()
* 特殊方法
    * __init__()
        * 初始化对象
    * __str__()
        * 打印对象时打印内容
* self参数
    * 实例引用名
    * 类方法的调用者和隐藏参数
* 隐藏数据
    * 查看或修改对象属性的方式有两种
        * 通过对象.属性名或对象.方法的方式
    * 隐藏修改对象属性的目的
        * 限制不合理的属性值
        * Python没有任何机制实现数据隐藏，如果需要，只能在编写代码时通过调用方法实现
