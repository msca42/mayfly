## 字典数据类型

- 字典是键值对的集合
- keys(): 返回 dict_keys 迭代键
- values(): 返回 dict_values 迭代值
- items(): 返回 dict_items 迭代键值对
- 通过 in 检查字典是否存在键或值
- get():
  - 参数 1:键
  - 参数 2:键不存在时返回值
  - 检查键是否在字典中
- setdefault():
  - 参数 1: 键
  - 参数 2: 当键不存在时要设置的值
  - 为字典键设置默认值
