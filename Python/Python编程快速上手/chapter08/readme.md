# 输入验证

- 学习使用第三方 PyInputPlus 模块进行输入验证

## PyInputPlus 模块

- 用于不同类型输入的函数
  - inputStr()、inputNum()
  - inputChoice()、inputMenu()
  - inputDatetime()、inputYesNo()
  - inputBool()、inputEmail()
  - inputFilepath
- 关键字参数
  - max、min、greaterThan 和 lessThan
  - blank=True 可以输入空格
  - limit、timeout、default
  - allowRegexes、blockRegexes
- 将自定义验证函数传递给 inputCustom()
