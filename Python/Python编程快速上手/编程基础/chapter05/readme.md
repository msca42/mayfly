# 字典和结构化数据
- 字典是键值对的集合，字典的索引称为键，通过索引可以访问值，字典中的项是不排序的

## 方法
- keys()
    - 返回值: dict_keys
    - 迭代键
- values()
    - 返回值: dict_values
    - 迭代值
- items()
    - 返回值: dict_items，本质是一个元组，包含键和值
    - 迭代键-值对
- get()
    - 参数: 要取得其值的键，当键不存在时返回的备用值
- setdefault()
    - 为字典的键设置默认值
