# 列表简介
- 列表的本质是一系列按照特定顺序排列的元素，使用[]表示
- 访问/修改元素
    - 通过索引index访问/修改列表元素，索引从0开始
- 添加元素
    - append(): 末尾追加
    - insert(): 在任意位置添加元素
- 删除元素
    - del语句
    - pop(): 删除末尾元素，通过参数可以删除任意位置的元素
    - remove(): 删除特定元素

- 排序
    - sort(): 永久排序
    - 函数sorted(): 临时排序
- 打印
    - reverse(): 反向打印
- 长度
    - len(): 获取列表长度
