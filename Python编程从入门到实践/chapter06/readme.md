## 使用字典

- 字典是一系列的键值对，使用{key:value}表示
- 通过索引可以访问或添加键值对
- 通过 del 可以删除键值对，指定字典名和要删除的键
- 通过 get()访问值，可以规避键不存在时报错的问题

## 遍历字典

- items()方法: 返回一个键值对列表
- keys()方法：返回所有的键
- values()方法: 返回所有的值
- sorted()函数: 临时排序
- set() 集合函数:
  - 每个元素都是唯一的
  - 通过{,,}可以直接创建集合
